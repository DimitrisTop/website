01 · Concept

Timalfes is a premium brand specializing in scientific cosmetology and pure apiary products. The goal was to build a clean, reliable digital storefront that reflects their high-quality physical products, without overcomplicating the technical setup.

02 · The Approach

The core decision was prioritizing business value over custom engineering. Instead of writing code from scratch, I selected established tools that allowed me to minimize development time and maximize focus on the underlying data architecture.

  • Platform Selection: Chose WordPress as a stable, scalable foundation that the client's internal team could easily adopt and manage post-handover.
  • Rapid Prototyping: Leveraged Elementor's modular system to quickly build, test, and iterate on the storefront layout without getting bogged down in syntax.
  • Logic Over Code: By reducing the time spent on basic web development, I was able to dedicate the project's focus to establishing a robust tracking setup and functional user flows.

03 · My Contribution

My role involved handling the basic web development, the visual setup, and solving specific user-flow problems:

  • Checkout Optimization: To reduce friction during purchases, I implemented a custom PHP script so that corporate billing details (invoice fields) are hidden by default and only appear via a dropdown if the user actively selects the invoice option.
  • Storefront Setup: Built and customized the page layouts using Elementor, ensuring the design matched the brand's premium identity.
  • Tracking & Feeds: Configured Google Tag Manager (GTM) and set up XML product feeds to connect the site's inventory with social media marketing campaigns.

04 · Handover & Status

This was a fixed-scope digital transformation project. Following deployment, I structured the handover documentation to ensure the internal team could manage the platform independently — covering the admin workflows, tracking configuration, and catalog maintenance process.

Note: Active technical support concluded in early 2025. Current site performance reflects post-handover management.
